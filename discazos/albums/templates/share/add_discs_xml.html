Paso 2 de 5: Agregar discos al álbum

El siguiente paso consiste en agregar el disco de audio (o los discos, si tiene 2 o 3) al álbum. Para lograr simplificar este paso, hicimos una aplicación de escritorio para ayudarte a completar esta tarea. La aplicación se encargá de generar 2 archivos: Un archivo de texto conteniendo la información básica de los discos y otro archivo MP3 con el álbum completo listo para ser subido (en el último paso de este asistente).

Descargue aquí la aplicación Discazos Creator (Ubuntu únicamente) y siga al asistente para generar estos archivos, basados en tus archivos MP3 del album.

Una vez terminado, suba el archivo XML generado por el programa que contiene la información básica de los discos del álbum, en el formulario que se encuentra aqúi debajo.
